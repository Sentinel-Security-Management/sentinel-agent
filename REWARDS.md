## Contributors to Sentinel Agent are eligible for rewards based on the impact of their contributions. Rewards are distributed according to the following criteria:

Minor Docs PR
→ Recognition

Bug Fix
→ 1 month Starter credits

Major Feature
→ 3 months Enterprise credits

Critical Security Finding
→ 6–12 months Enterprise credits

Major Architecture Contribution
→ Lifetime Contributor Tier


Founding Contributors
├─ Lifetime Enterprise Sandbox
├─ Private Discord/Slack access
├─ Early feature access
├─ Name on Contributors page
└─ Direct maintainer communication channel
